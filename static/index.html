CNN Model Architecture

A sequential convolutional neural network with ~93K trainable parameters.

Layer Pipeline

Input
Input Layer

Conv2D
Conv2D — 32 filters, 3×3, ReLU
Extracts low-level features (edges, textures) → 26×26×32
MaxPool
MaxPooling2D — 2×2
Downsamples spatial dimensions → 13×13×32
Conv2D
Conv2D — 64 filters, 3×3, ReLU
Extracts complex high-level features → 11×11×64
MaxPool
MaxPooling2D — 2×2
Further downsampling → 5×5×64
Flatten + Dense
Flatten → Dense (64 units, ReLU)
Converts feature maps to 1D vector (1600) and learns combinations
Output
Dense — 10 units, Softmax
Outputs probability distribution across 10 fashion classes
